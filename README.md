# Arduino-CSF

Arduino Unit based on Canyon Crest Academy's 2022 CSF class. 

Each folder name is dependent on the assignment's name in the class. 

The final project folder name is: U5_08b_Combo_Lock_project

The Arduino Uno was used in a variant of C++. 

If interested in an online simulation, I used this Tinkercad website: https://www.tinkercad.com/things/flHpOFbJ1CN-arduino-simulator-and
